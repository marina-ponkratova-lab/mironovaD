## Описание программы

Training Planner — простое приложение для планирования и учёта тренировок с графическим интерфейсом. Позволяет добавлять тренировки с указанием даты, типа и длительности, фильтровать записи по этим параметрам. Данные сохраняются в формате JSON.

## Как использовать

1. Установите Python 3.x.
2. Скопируйте файлы проекта в одну папку.
3. Запустите main.py.
4. Введите дату (ГГГГ-ММ-ДД), выберите тип тренировки и длительность (мин), нажмите «Добавить тренировку».
5. Для фильтрации используйте вкладку «Фильтр».
6. Данные сохраняются автоматически.

## Примеры использования

- Добавить тренировку: 2026-04-24, Кардио, 45 мин.
- Отфильтровать тренировки по типу «Силовая».
- Посмотреть все тренировки за определённую дату.
